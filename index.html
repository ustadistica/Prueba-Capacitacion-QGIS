¿Qué es QGIS?

Imagina un "Photoshop" o un "Excel" pero para mapas. ¡Eso es QGIS! Es un Sistema de Información Geográfica (SIG) profesional, gratuito y de código abierto.

Te permite visualizar, gestionar, editar y analizar datos geográficos para crear mapas increíbles. Al combinarlo con Python, ¡puedes automatizar tareas y realizar análisis complejos que serían imposibles a mano!

1. Fundamentos de GeoPandas

Este cuaderno es tu puerta de entrada. Aprenderás a manipular datos geográficos en Python con GeoPandas, la librería esencial que combina el poder de Pandas con la capacidad de entender formas y ubicaciones.

  • Carga de Datos Vectoriales: Lee archivos `.shp` y explora sus geometrías.
  • Visualización de Mapas: Crea tus primeros mapas estáticos de forma sencilla.
  • Sistemas de Coordenadas (CRS): Entiende por qué son cruciales para la precisión de tus mapas.

2. Geocodificación de Direcciones

En este caso práctico, transformarás direcciones de texto (como "Calle 13 # 4-20") en puntos sobre un mapa. Una habilidad clave para logística, análisis de mercado o cualquier proyecto que necesite ubicar lugares.

  • APIs de Geocodificación: Usa `geopy` para conectar con servicios que convierten texto a coordenadas.
  • Limpieza y Estandarización: Prepara tus datos de direcciones para obtener los mejores resultados.
  • Mapas Interactivos: Visualiza tus resultados en un mapa dinámico con `Folium` donde podrás hacer zoom y explorar.
